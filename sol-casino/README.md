# $SOL Casino - Dice Roll Game

A decentralized dice roll betting game built on Solana blockchain using Anchor framework.

## Project Structure

```
sol-casino/
├── programs/
│   └── sol-casino/
│       └── src/
│           └── lib.rs          # Main program logic
├── tests/                       # Integration tests
├── app/                         # Frontend (to be added in Sprint 3)
├── Anchor.toml                  # Anchor configuration
└── Cargo.toml                   # Rust workspace config
```

## Setup

1. Install dependencies:
```bash
npm install
```

2. Build the program:
```bash
anchor build
```

3. Run tests:
```bash
anchor test
```

## Development

This project uses Solana Devnet for development and testing.
